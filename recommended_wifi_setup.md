# Recommended WiFi setup
---

In this tutorial, we are going to use the *ESP8266* WiFi module to communicate with *Pixhawk* via Wifi.

Required:
* [ESP8266 module](https://www.sparkfun.com/products/13678)
* [Logic level converter](https://www.sparkfun.com/products/12009) OR [XBee breakout board](http://www.robotshop.com/en/droids-xbee-simple-board.html)
